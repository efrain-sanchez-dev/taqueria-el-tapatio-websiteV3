Taqueria El Tapatio Website Project

This project is a website created for **Taqueria El Tapatio**, a Mexican restaurant located in Farmersville, California. The purpose of the website is to provide customers with restaurant information, menu access, and an engaging online experience that reflects the restaurant’s atmosphere and branding.

[https://efrain-sanchez-dev.github.io/taqueria-el-tapatio-websiteV3/](https://efrain-sanchez-dev.github.io/taqueria-el-tapatio-websiteV3/)

The website was developed using the following technologies: HTML, CSS and JavaScript

Features Included: Responsive website layout, Restaurant branding and visuals, Navigation menu, Interactive website elements using JavaScript, Organized sections for customer information, Mobile-friendly design
